# 观微 (Lucid) 核心算法设计参考

观微系统彻底摒弃了传统志愿填报软件中“冲、稳、保、垫”全凭专家感性经验分类的方式。本系统将志愿填报问题抽象为一个在不完全信息博弈下的概率估计与风险控制问题，通过高斯概率积分函数（CDF）将每一个志愿的选择量化为具有精确数学意义的“录取胜率”。

## 1. 录取概率的高斯正态分布建模

我们将某高校在某省份某科类往年的最低录取位次（提档线） $R_{cutoff}$ 建模为一个服从正态分布（Gaussian Distribution）的随机变量：
$$R_{cutoff} \sim \mathcal{N}(\mu, \sigma^2)$$

### 参数估算
对于一所高校的特定专业：
- **均值 $\mu$**：基于该校近 3 年的平均提档线等效位次：
  $$\mu = \frac{1}{k}\sum_{i=1}^{k} R_{cutoff, i}$$
- **标准差 $\sigma$**：反映了提档线的历史波动剧烈程度。若历史波动极小，为防止计算出的概率极端化导致系统失去容错性，我们会引入保护性偏置：
  $$\sigma = \max\left(\text{StdDev}(R_{cutoff}), \sigma_{min}\right)$$
  其中 $\sigma_{min}$ 设定为 50，以保证即使提档线非常稳定的学校，其计算出的胜率依然保持平滑过渡。

### 胜率公式
对于等效位次为 $R_{user}$ 的考生，其被该校录取的概率为该校提档位次大于等于考生等效位次的概率，即：
$$P(Accept) = P(R_{cutoff} \ge R_{user})$$

通过正态分布的累积分布函数（Cumulative Distribution Function, CDF） $\Phi$，我们可以表示为：
$$P(Accept) = \Phi\left(\frac{\mu - R_{user}}{\sigma}\right)$$

在没有安装科学计算库（如 `scipy`）的纯 Python 运行环境下，我们采用高精度的数值逼近算法（Abramowitz and Stegun 逼近）来计算标准正态分布的累积概率，其最大绝对误差小于 $1.5 \times 10^{-7}$：

```python
def std_normal_cdf(x: float) -> float:
    """Standard normal cumulative distribution function (A&S approximation)"""
    if x < 0:
        return 1.0 - std_normal_cdf(-x)
    
    # Constants
    b1 = 0.319381530
    b2 = -0.356563782
    b3 = 1.781477937
    b4 = -1.821255978
    b5 = 1.330274429
    p = 0.2316419
    c = 0.39894228
    
    t = 1.0 / (1.0 + p * x)
    exponent = -0.5 * x * x
    # Prevent overflow in extreme ranges
    if exponent < -350.0:
        return 0.0 if x < 0 else 1.0
        
    prob = 1.0 - c * math.exp(exponent) * t * (t * (t * (t * (t * b5 + b4) + b3) + b2) + b1)
    return prob
```

## 2. 志愿梯度与胜率映射

系统根据计算出的绝对胜率 $P(Accept)$，将志愿自动划分为四个严格的梯度。这四个梯度为考生的志愿编排提供了严密的博弈理论支撑：

| 梯度区间 | 英文标识 | 胜率范围 | 推荐数量 | 博弈论/决策论释义 |
| :--- | :--- | :--- | :--- | :--- |
| **激进冲刺区** | `Reach` | $15\% \le P < 45\%$ | 2 - 3 个 | 属于小概率事件，适合博弈捡漏。适合填报往年爆冷门或处于上涨周期开始回落的学校。 |
| **合理匹配区** | `Match` | $45\% \le P < 75\%$ | 3 - 4 个 | 属于期望概率区，是“纳什均衡”的最优主阵地。考生的分数与这些学校的定位高度契合。 |
| **安全保底区** | `Safety` | $75\% \le P < 95\%$ | 2 个 | 属于极高概率区，代表考生相比于这些学校的历年录取，占据了绝对主动权。 |
| **极限托底区** | `Fall-back` | $P \ge 95\%$ | 1 - 2 个 | 作为最后一道防线，胜率近乎 100%，确保即使出现极端报考异常情况也绝不滑档。 |

### 🚫 硬性拦截规则
对于 $P(Accept) < 15\%$ 的学校，系统在粗排阶段进行“硬性拦截剔除”。禁止向考生推荐这类无法企及的极低胜率高校，从而合理利用宝贵的志愿名额。
